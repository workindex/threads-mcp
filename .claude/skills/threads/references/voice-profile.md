# 보이스 프로필 (측정 기반) — Layer B 골격

> 작성자님 문체의 **정량 골격**. 주관 묘사가 아니라 코퍼스에서 관측한 수치.
> 출처: `get_draft_context` 성공글 Top20(좋아요+댓글×3). 측정 v1·근사 (2026-05-30). 코퍼스 성장 시 갱신.
> 근거(Patel et al., 2023): 정량 프로필 + 2~5개 예문 = 두 방법 단독보다 우수. 주관 묘사("직설적이다")는 LLM 출력에 거의 효과 없음. → **이 파일은 골격, 예문이 질감.**

## 어미 — 3개 레지스터 (한 글 안에서 섞지 않음)

소재로 레지스터를 고른다:
- **습니다체** (~50%) — 공유·강의·사업 회고. "~되더라고요", "~같습니다", "~될 겁니다"
- **한다/했다체** (~30%) — 서사·선언. "그냥 만들었다", "전적으로 동의했다", "매일 사라진다"
- **감정 단편** (~20%) — 말줄임으로 흘림. "억울하... 아니 아깝다...", "ㅎ;;;;", "미루지 말자!"

## 길이 — 2모드 (중간 거의 없음)

- **초단문**: 1~3줄. 질문·관찰·공지. (예: 5·6·15·19위)
- **중간 서사**: 150~400자. 경험→깨달음. (예: 8·9·17위)
- 400자 초과 거의 없음. 길면 자른다.

## 첫 줄(훅) — 4패턴

| 패턴 | 예 |
|------|-----|
| 대비·질문 | "성공 사례는 넘쳐나는데, 실패 사례 모아놓은 곳은 왜 없을까?" |
| 구체 수치 | "하루 6시간씩, 2명이서", "약 6천만원짜리" |
| 평서 사실 | "안티그래비티에 클로드코드 확장 프로그램을 설치할 수 있습니다." |
| 라벨형 제목 | "비개발자의 용어 정리", "B2B/G 7년차에 느낀 점" |

## 종결 — 청중 질문 빈도 높음

- Top20 중 ~30%가 질문으로 끝남: "관심 있으신 분 계신가요?", "있나요?", "알려주세요"
- 공지·모임글은 거의 항상 질문 종결.

## 구조 템플릿 — 3형

- **그냥 만들었다형**: 문제 발견 → "없어서 그냥 만들었다" → 컨셉/오퍼
- **서사→깨달음형**: 경험 → "아직도 정답을 모르겠지만" → 한 줄 교훈(종종 따옴표 인용)
- **수치 폭로형**: 6천만원 → 600만원처럼 숫자 대비로 통찰

## 리듬·표기

- 줄바꿈 잦음 — 한 생각 = 한 덩어리, 짧은 단락.
- 말줄임표 "..." ~20~25% 글에서 감정·불확실 표현.
- 이모지 거의 0. 써도 1개.
- 자기비하 마커 허용: "ㅎ;;;;", "ㅎ;".

## 어휘

- 구어체 + 약어(클로드·옵클·바이브코딩) 자연스럽게.
- 전문용어는 쓰되 바로 풀어줌 (brand_dna 금기: 지나친 전문용어).
